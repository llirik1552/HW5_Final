Команды Git

  • git init - инициализация репозитория.
  • git clone ссылка с GitHub - клонирование удаленного репозитория с GitHub на свой компьютер.
  • git add index.html - добавить репозиторий в отслеживаемые файлы.
  • git status - проверить статус файла.
  • git commit -m "name" - сделать коммит.
